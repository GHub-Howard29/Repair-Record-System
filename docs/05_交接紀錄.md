- 確認 Google Drive 縮圖備援在授權帳號的 GitHub Pages 部署站可讀取；若無法讀取，需改為受權限保護的後端預覽端點。
- 驗證送回日期儲存後，統計卡數量與「已完成」清單立即一致；並驗證查詢條件與統計卡手動篩選的優先順序。
- 在手機窄版檢查附件、收費摘要、歷史維修與功能選單的捲動及操作。

### 待辦事項

1. 補齊自動化測試：製造號碼格式、完成後鎖定、統計／查詢篩選、附件預覽合併與同步失敗重試。
2. 驗證 Firestore Rules 的刪除拒絕規則，並確認 GitHub Actions Secrets 與正式部署環境一致。
3. 視實機結果決定 Drive 預覽採縮圖備援或新增 Firebase Functions 預覽端點。
4. 處理 Vite bundle 大小警告：評估 Firebase 與匯出功能的 lazy loading。
